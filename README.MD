# Portfolio Personal

Portfolio web desarrollado para presentar mi perfil profesional, habilidades y proyectos como diseñador y desarrollador web.

## Características

* Diseño responsive
* Interfaz moderna y minimalista
* Sección de proyectos destacados
* Presentación profesional
* Formulario de contacto

## Tecnologías

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## Instalación

```bash
git clone https://github.com/usuario/portfolio.git
cd portfolio
```

Abrir `index.html` en el navegador.
